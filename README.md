# TaskManager

TaskManager es una aplicación de ejemplo desarrollada con .NET 8 y siguiendo principios de Clean Architecture y CQRS. El objetivo es construir una base sólida y extensible para gestionar tareas de trabajo, integrando características como paginación, filtrado y ordenación dinámica.
